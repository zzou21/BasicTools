Basic Tools

Version 1.3.37

An open-source, free, and minimum-computing suite of everyday computer utilities designed to run locally on your machine.

Visit Basic Tools' website to share feedback, report bug, get in contact, suggest a new tool to be added, get more info, or download the newest version of Basic Tools.


Basic Tools allows users to perform mundane yet necessary functionalities without the need of AI agents, AI programs, cloud storage spaces, proprietary (and often costly) softwares, internet connection, and user account creation. It is a tool designed to honor human independence and human actions in the digital space while addressing the lack of attention in the software industry toward creating better user experience.

The tools included in this app are so basic that you can use them in a variety of scenarios, such as professional, research, academic, or personal tasks. When I was learning coding for the first time, I asked the person who opened the world of programming to me the question "Could you please demonstrate what specific real-life situations would we use an accumulator variable?" and got the answer "Everywhere you see." I hope Basic Tools addresses some of the most mundane, basic, yet vexing utilities and user experience gaps today such that it is the "everywhere you see" tool for your work on your laptop.

I welcome any recommendations or feedback. Please get in contact with me to share your thoughts (e.g., a new mundane function that will save you so much time that you'd like to see on Basic Tools) via the link to the website above.

Project Inception

Scenario 1: I use a MacBook. The MacBook search bar consistently underperforms (especially the search function for a specific directory).

Scenario 2: I take a lot of photos, especially when going on archival research trips. I need a way to easily convert them to different formats and name the photo files in specific ways. There does not exist an efficient workflow that doesn't require expensive Adobe subscription or AI agent.
