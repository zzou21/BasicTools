PDF Merger

Combine multiple PDFs into one. Drag to reorder.

1. Select PDF Files

To select multiple files, press and hold the CONTROL (or COMMAND, for Mac users) key and select your files. You should see all your selected photos highlighted in your file explorer window.

You can press the "Choose PDFs" button multiple times to add files that you have forgotten to include or to merge files coming from different folders.

After selecting your files, drag the icon for each file to rearrange them into the order that you'd like your final merged PDF to look like.

3. Output File Name
